#### 调用sed 的方式

sed 会把处理后的结果直接通过stdout 进行输出。

```bash
# 通过管道调用
# 通过管道调用，会把每个接收到的行进行处理
echo abc | sed -r 's###g'

# 其中 -r 的选项表示采用正则表达式进行处理
# s 表示替换
# sed 中的三个#分割了两个部分一个是需要匹配的模式串，另一个是匹配之后的需要替换成的模式串。

# 通过文件调用
```

**注意点**

- **sed的正则表达式采用的是linux 的标准，即分组的结果引用采用的是\1，而不是\$1**

- 表示指定选项的数量在sed中是用 {}表示。 {0,}表示\*，{1,} 表示 +
