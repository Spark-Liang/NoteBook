### 安装基础软件

- 修改国内源

#### 修改国内源

1. 备份源列表
   
   ```bash
   mv /etc/apt/sources.list  /etc/apt/sources.list.bak, 备份原来的自带的源
   sudo vi /etc/apt/sources.list, 将上面的源挑选一个黏贴进文本，保存
   sudo apt-get update
   ```
